# Layer 1 IT Assurance

- [[sources/verizon-process-101-layer1-it-assurance-gery-transcript|verizon process 101 layer1 it assurance gery transcript]]: Layer 1 IT Assurance is the post-Verizon handoff validation process for preparing sites for migration by configuring access, collecting support inventory, updating monitoring/authentication systems, and verifying physical connectivity.
