# Verizon Migration Procedures

- [[sources/verizon-process-101-mop-cutsheet-creation-transcript|verizon process 101 mop cutsheet creation transcript]]: Verizon migration procedures cover creating cut sheets from I specs, adapting generic or FOA-derived MOPs, validating maintenance-window steps, and adjusting strategy by platform and site type.

- [[sources/verizon-process-101-ispec-review-james-e-gery|verizon process 101 ispec review james e gery]]: ISPEC review procedures include downloading and flattening I-spec spreadsheets, checking hidden tabs, validating D1/D2 access, reconciling SFP and port assignments, and documenting old-to-new router mappings for cut sheets and install notes.
- [[sources/verizon-process-101-layer1-it-assurance-gery-transcript|verizon process 101 layer1 it assurance gery transcript]]: Layer 1 validation procedures include collecting I spec details, configuring management access, confirming software and hardware state, adding new devices to monitoring and authentication systems, and verifying new-to-legacy cabling through an approved non-service-impacting KIRKE.
