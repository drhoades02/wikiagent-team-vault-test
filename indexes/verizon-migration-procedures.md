# Verizon Migration Procedures

- [[sources/verizon-process-101-mop-cutsheet-creation-transcript|verizon process 101 mop cutsheet creation transcript]]: Verizon migration procedures cover creating cut sheets from I specs, adapting generic or FOA-derived MOPs, validating maintenance-window steps, and adjusting strategy by platform and site type.

- [[sources/layer1-tasks|layer1 tasks]]: Verizon Layer 1 route/switch procedures include downloading the relevant ISPEC, configuring new devices, verifying hardware, software, access, archival, monitoring integrations, and connectivity before declaring the site ready for migration tasks.
